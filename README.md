# 集团月度经营分析看板

该项目为“月度经营看板”原型应用，包含四个页面：
- 总体经营情况
- 渠道运营情况
- 客户运营情况
- 产品运营情况

内置示例数据与口径，支持按月份查看关键指标：毛利率、本年累计、去年同期、同比、本年预算、预算完成率、零售业务周转率(次)、海南区域营收(亿)、线上完税GMV(亿)、国际化业务占比、国潮商品销售额(亿)等。可用于对接集团数据集成平台的落地实现与二次开发。

## 快速开始

1) 安装依赖

```bash
python3 -m venv .venv && source .venv/bin/activate
python -m pip install -U pip
pip install -r requirements.txt
```

2) 运行应用

```bash
streamlit run app.py
```

3) 浏览器访问输出的本地地址即可（默认 http://localhost:8501 ）。

## 目录结构

```
/workspace
  ├─ app.py                # Streamlit 应用入口（四个页面）
  ├─ requirements.txt      # 依赖
  ├─ README.md             # 本文件
  └─ data/                 # 示例数据（可替换为真实数据源）
       ├─ monthly_kpi.csv
       ├─ channel_kpi.csv
       ├─ customer_kpi.csv
       └─ product_kpi.csv
```

## 数据接入与口径

- 现阶段应用默认读取 `data/*.csv` 示例数据，字段包含：
  - 总体：营业收入、净利润、归母净利润、净资产收益率(ROE)、毛利率、海南区域营收、线上完税GMV、国际化业务占比、国潮商品销售额、零售业务周转率、预算（收入/利润）等；
  - 渠道：按渠道分组的收入、毛利率、预算与同比；
  - 客户：活跃客户、新增客户、复购率、ARPU 等；
  - 产品：按品类分组的收入、毛利率与国潮商品标识等。
- 应用中统一计算：
  - 本年累计（YTD）：按自然年从 1 月累加至所选月份；
  - 去年同期：与所选月份同月上一年；
  - 同比： (本期 - 去年同期) / 去年同期；
  - 预算完成率： 实际 / 预算。

## 替换为真实数据

- 可将 `data/*.csv` 替换为真实导出数据；
- 或设置环境变量 `DATABASE_URL`（如 `postgresql://user:pass@host:5432/db`），应用将尝试从数据库读取相同维度与字段（可在代码中查看字段要求）。

## 注意

- 示例数据仅用于前端展示与联调演示，口径以集团数据治理口径为准；
- 页面与字段均支持扩展，可按三级公司、区域、事业群等维度增加筛选器与钻取；
- 若需对接集团统一认证、权限与指标平台，请在此基础上接入对应 SDK 与 API。
