# 集团月度经营看板

一个现代化、可交互的数据可视化大屏系统，为集团数据集成平台提供实时的月度经营分析展示。

## 🎯 项目背景

为进一步强化集团数据集成平台指标体系的完整性与实用性，本项目由中免牵头开发重点三级公司数据看板，旨在：

- **解决数据分散问题**：整合三级公司数据，打破数据孤岛
- **提升监控时效性**：实现实时数据可视化呈现
- **优化决策效率**：为集团战略决策提供量化依据
- **统一分析标准**：避免各事业群数据口径不一的问题

## ✨ 功能特性

### 📊 四大核心页面

1. **总体经营情况**
   - 营业收入、净利润、归母净利润、净资产收益率
   - 月度收入和利润趋势图表
   - 关键业务指标（零售业务周转率、海南区域营收、线上完税GMV等）

2. **渠道运营情况**
   - 线下门店、线上平台、批发渠道全覆盖
   - 渠道营收趋势和占比分析
   - 坪效、人效、获客成本等效率指标

3. **客户运营情况**
   - 客户总数、活跃客户、新增客户、留存率
   - 客户价值分布和生命周期分析
   - 客单价、复购率、购买频次等行为指标

4. **产品运营情况**
   - 香化美妆、精品服饰、食品饮料等产品线分析
   - 产品销售趋势和毛利率对比
   - 国潮商品专区数据展示

### 🎨 设计特色

- **深色主题**：适合大屏展示的专业视觉效果
- **渐变配色**：紫色主调配合多彩渐变，视觉层次丰富
- **卡片布局**：模块化设计，信息组织清晰
- **动态效果**：页面切换动画、数字滚动效果、悬停交互
- **响应式设计**：支持不同屏幕尺寸的完美适配

### 🎮 交互功能

- **页面切换**：点击导航或使用键盘快捷键（1-4数字键）
- **实时时间**：头部显示当前日期时间
- **数据刷新**：支持手动刷新数据
- **键盘支持**：方向键导航、F11全屏、F5刷新
- **图表交互**：Chart.js驱动的动态图表，支持悬停详情

## 🚀 快速开始

### 本地运行

1. 克隆项目到本地
```bash
git clone [项目地址]
cd dashboard
```

2. 使用本地服务器运行（推荐）
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx http-server

# 使用PHP
php -S localhost:8000
```

3. 访问 `http://localhost:8000` 查看看板

### 直接访问

也可以直接双击 `index.html` 在浏览器中打开，但推荐使用HTTP服务器以获得最佳体验。

## 📱 使用说明

### 导航操作

- **鼠标点击**：点击顶部导航菜单切换页面
- **键盘快捷键**：
  - `1-4`：快速切换到对应页面
  - `←/→`：上一页/下一页
  - `F11`：全屏模式
  - `F5`：刷新数据

### 数据展示

- **指标卡片**：显示核心经营指标，包含同比增长、预算完成率等对比数据
- **趋势图表**：展示时间序列数据，支持多维度对比
- **占比分析**：饼图、环形图展示结构性数据
- **实时更新**：点击右上角刷新按钮更新数据

## 📊 数据结构

### 核心指标维度

- **财务指标**：营业收入、净利润、归母净利润、净资产收益率
- **业务指标**：毛利率、零售业务周转率、国际化业务占比
- **时间维度**：本年累计、去年同期、同比增长、预算完成率
- **区域指标**：海南区域营收、线上完税GMV
- **产品指标**：国潮商品销售额、各产品线占比

## 🎨 自定义配置

### 主题配色

可以在 `styles.css` 中修改CSS变量来调整主题：

```css
:root {
  --primary-color: #7877c6;
  --secondary-color: #ff77c6;
  --accent-color: #78ffc6;
  --background-dark: #0f1419;
  --surface-color: #1a2332;
}
```

### 图表配置

在 `script.js` 中的 `getChartOptions()` 函数可以统一调整图表样式和行为。

### 数据源

当前使用模拟数据，实际部署时需要：

1. 修改 `refreshData()` 函数连接真实API
2. 更新各页面的图表数据源
3. 配置数据刷新频率和缓存策略

### 生产环境部署

## 📈 数据指标说明

### 总体经营指标

- **营业收入**：企业在特定期间内的主营业务收入总额
- **净利润**：扣除所有费用后的企业纯利润
- **归母净利润**：归属于母公司股东的净利润
- **净资产收益率**：净利润与股东权益的比值，衡量盈利能力

### 渠道运营指标

- **坪效**：每平方米营业面积产生的营业收入
- **人效**：每名员工产生的营业收入
- **获客成本**：获得一个新客户所需的平均成本
- **转化率**：访客转化为实际购买客户的比例

### 客户运营指标

- **客单价**：每次交易的平均金额
- **复购率**：在特定期间内重复购买的客户比例
- **客户留存率**：在特定期间内保持活跃的客户比例
- **生命周期价值**：客户在整个关系期间的预期价值
